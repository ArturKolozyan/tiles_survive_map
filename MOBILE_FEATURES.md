# 📱 Мобильная адаптация Battle Map

## Что было сделано

### 1. Адаптивный сайдбар
- На мобильных устройствах (< 768px) сайдбар скрыт по умолчанию
- Кнопка меню (☰) в левом верхнем углу для открытия
- Сайдбар выезжает слева с плавной анимацией
- Затемнение фона при открытом сайдбаре
- Автоматическое закрытие при выборе режима

### 2. Touch-управление картой
- **Одним пальцем**: перетаскивание карты
- **Двумя пальцами**: pinch-to-zoom (масштабирование)
- **Тап по точке**: выбор точки и открытие информации
- Оптимизированные размеры кнопок (минимум 44px для touch)

### 3. Оптимизированный интерфейс
- Карта занимает весь экран на мобильных
- Карточка информации о точке выезжает снизу
- Модальные окна адаптированы под маленькие экраны
- Увеличенные шрифты для лучшей читаемости
- Предотвращение зума при фокусе на input (font-size: 16px)

### 4. Ландшафтная ориентация
- Сайдбар остается слева в landscape режиме
- Карточка точки справа (как на десктопе)
- Оптимальное использование пространства

### 5. Улучшения UX
- Уведомления смещены вниз чтобы не перекрывать кнопку меню
- Кнопка центрирования карты всегда доступна
- Плавные анимации и переходы
- Закрытие карточки точки при клике вне её

## Как тестировать

### На реальном устройстве
1. Откройте приложение на телефоне
2. Нажмите кнопку ☰ для открытия меню
3. Попробуйте перетаскивать карту одним пальцем
4. Используйте pinch для масштабирования
5. Тапните по точке для просмотра информации

### В браузере (Chrome DevTools)
1. Откройте DevTools (F12)
2. Включите Device Toolbar (Ctrl+Shift+M)
3. Выберите мобильное устройство (iPhone, Android)
4. Протестируйте все функции

## Поддерживаемые разрешения
- 📱 Мобильные: 320px - 768px
- 📱 Планшеты: 768px - 1024px
- 💻 Десктоп: 1024px+
