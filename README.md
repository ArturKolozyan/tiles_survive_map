# Tiles Survive - Battle Map

Интерактивная карта для планирования битв за территории в игре Tiles Survive.

## Возможности

- Создание карт с точками (вышки, логова, точки начала)
- Соединение точек дорогами
- Автоматическое расширение территорий каждый день
- Система боев за спорные территории
- Подсчет добычи нефти
- Таймер карты
- Метки для планирования

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Запустите сервер:
```bash
python manage.py runserver
```

3. Откройте браузер: http://127.0.0.1:8000/

## Использование

### Режим разработки
1. Добавляйте точки на карту
2. Соединяйте их дорогами
3. Настройте параметры (нефть, день открытия)
4. Выберите свою точку начала
5. Установите дату старта и запустите карту

### Режим игры
1. Отмечайте результаты боев (захвачено/бой)
2. Ставьте метки для планирования
3. Следите за статистикой и таймером
4. Обновляйте фронт кнопкой "Обновить фронт"

## Игровая механика

- Каждый день территории расширяются от точек начала
- Если границы игрока и врага встречаются - начинается бой (красная точка)
- Захваченные точки (зеленые) приносят нефть
- Вражеские границы соединяются мирно
- Закрытые точки открываются в указанный день
