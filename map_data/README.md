# Папка с картами

Здесь хранятся все карты в формате JSON.

## Формат файла карты

Каждая карта сохраняется в отдельный файл `map_{id}.json`:

```json
{
  "id": 1234567890,
  "name": "Название карты",
  "data": {
    "points": [...],
    "connections": [...],
    "userMarkers": [...]
  },
  "duration_days": 10,
  "start_time": "2026-02-25T00:00:00",
  "is_running": false,
  "created_at": "2026-02-24T20:00:00",
  "updated_at": "2026-02-24T20:00:00"
}
```

## Перенос данных

Чтобы перенести карты на другой компьютер:
1. Скопируйте всю папку `map_data/`
2. Вставьте в корень проекта на другом компьютере
3. Все карты будут доступны!

## Резервное копирование

Просто скопируйте эту папку в безопасное место.
Все ваши карты сохранены в читаемом JSON формате.
